xargs命令

发表于 2015-01-22   |  

xargs是一条Unix和类Unix操作系统的常用命令。它的作用是将参数列表转换成小块分段传递给其他命令，以避免参数列表过长的问题。通常用来批量处理。

先看几个示例。

####demo1：
命令：find . | head | xargs file
作用：find .查找当前目录下的文件，head取前5个，然后针对每个文件使用file命令来获取文件类型。
输出：

####demo2
命令：cat ip.txt | xargs -i host {}
作用：逐行输出ip列表，针对每个ip获取ip对应的主机名
输出：

####demo3
命令：cat ip.txt | xargs -p -i host {}
作用：demo2提示模式，输入y继续执行

####demo4
命令：find . -name '*.php' | xargs grep 'wangchuanzheng'
等价：grep ‘wangchuanzheng’ `find . -name ‘*.php’`
作用：查找php文件，筛选出包含wangchuanzheng字样的文件
输出：

##参数
-i : 使用{}替换前面输出的内容。
-p : 提示模式。确认是否执行后面的命令，y执行，n不执行。
