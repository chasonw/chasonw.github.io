cURL命令

发表于 2014-09-01   |   分类于 linux   |  

linux环境下开发有时需要请求http接口进行调试，而在命令行下cURL是一款请求http的利器。
实际上，cURL并不单纯为http而生，从官网http://curl.haxx.se的介绍可以知道：curl是一款使用URL语法传输数据的命令行工具，支持FTP, FTPS, Gopher, HTTP, HTTPS等等协议，支持SSL认证。HTTP协议支持POST、PUT、cookie等等。而web开发者通常使用的只有HTTP。

常用命令

下面是一些常用命令（也可通过curl --help查看）：

  • 请求http，获取源码，默认GET方式

  • 指定method请求：使用-X，支持POST/PUT/DELETE等等

  • 指定POST参数：使用-d

  • 指定header参数

  • 显示返回报文头部

  • 调试http交互过程

  • 指定referer、UA、cookie

变量

FORMAT中的%{variable_name}均为变量，有以下几种跟时间相关的变量（均以秒为单位）：

  • time_namelookup : 从一开始到域名解析完成时所耗时间
  • time_connect : 从一开始到TCP连接远程主机完成时所耗时间
  • time_pretransfer : 从一开始到文件传输准备就绪所耗时间
  • time_starttransfer : 从一开始到第一字节准备开始传输，包括time_pretransfer和服务器计算时间
  • time_total : 从一开始到请求结束时间

  • receiving = time_total - time_starttransfer （客户机从服务器下载数据所用的时间）
