云盘git备份代码

发表于 2015-12-25   |  

GitHub就是一个免费托管开源代码的远程仓库，但有时候我们不想公开源代码，同时想要找个安全靠谱的地方备份维护自己的代码。
这种情景下，我们就可以在云盘上搭建一台Git服务器作为私有仓库使用。
